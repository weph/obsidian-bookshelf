# Getting Started

## Installation

- Go to "Settings" > "Community plugins" and make sure "Restricted mode" is turned off
- Search for "Bookshelf" in Obsidian's community plugins browser
- Install and enable the plugin
- Open the plugin settings to customize as needed

## Creating your first book note

By default, Bookshelf looks for your book notes in the **Books** folder in your vault's root.

```markdown
---
cover: '[[test-driven-development-by-example.jpg]]'
author:
    - '[[Kent Beck]]'
published: 2002-01-01
tags:
    - computer-science
    - programming
    - tdd
rating: 5
pages: 216
lists:
    - Programming Books
    - Testing
comment: Probably the best introduction to TDD ever written
---

# Test-Driven Development by Example

## Reading Journey
```

:::tip

You can find more examples in the [demo vault](https://github.com/weph/obsidian-bookshelf/tree/main/resources/demo-vault).

:::
