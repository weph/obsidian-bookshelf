# Bases Views

If you want more control over which notes and data are displayed, you can use Bookshelf's Bases views.

Create a Base and select either the **Bookshelf gallery** or **Bookshelf table** layout.

![Layout selection](/img/bases-layouts.png)

Each layout comes with a set of Bookshelf-specific fields that you can show or hide as needed.
You can also add additional fields through Bases' Properties selection, just like in any other Bases view.
