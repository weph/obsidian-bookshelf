# About Bookshelf

Bookshelf is a plugin for [Obsidian](https://obsidian.md/) that helps you manage books, track reading activity in your
notes, explore reading statistics, and browse your library through dedicated views.

The plugin includes built-in library and statistics views that work out of the box. For users who want more control,
Bookshelf also integrates with [Obsidian Bases](https://obsidian.md/help/bases), making it easy to build custom views.
